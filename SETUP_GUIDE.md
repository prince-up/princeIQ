### Step 1: Get OpenAI API Key (FREE to start)

1. Go to: **https://platform.openai.com/signup**
2. Create a free account (sign up with Google/Email)
3. Add payment method (pay-as-you-go, not subscription)
   - You get $5 free credits
   - After that: ~$0.001 per explanation (super cheap!)
4. Go to: **https://platform.openai.com/api-keys**
5. Click "Create new secret key"
6. Copy the key (starts with `sk-...`)

### Step 2: Configure PrinceX

1. **Right-click** the PrinceX extension icon in Chrome
2. Click **"Options"**
3. Paste your API key
4. Choose model: **GPT-4o Mini** (recommended - fast & cheap)
5. Click **Save Settings**

### Step 3: Use It!

1. Go to any website (GitHub, Stack Overflow, etc.)
2. **Highlight** any text (e.g., "recursion", "polymorphism")
3. **Click** the PrinceX extension icon
4. Get instant AI explanation! ðŸŽ‰
